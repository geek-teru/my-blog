### 環境準備
1. Node.js / npm のインストール
2. Gatsby CLI のインストール
```
npm install -g gatsby-cli
```

3. Gatsby プロジェクトの作成
```
# Gatsby のスターターテンプレートを利用してブログ作成
npx gatsby new my-blog https://github.com/gatsbyjs/gatsby-starter-blog

# npm パッケージをインストール
npm install

# ローカルサーバーの起動
gatsby develop

# ブラウザで確認
http://localhost:8000/

```

# GraphQL の確認
```
http://localhost:8000/___graphql
```
